## Overview

This repository contains a backend API and a frontend dataset explorer. The homepage displays categories and procedures returned by the backend in a searchable table.

## Running the project

- Start the backend from `backend/` with `npm install` and `npm run dev`
- Start the frontend from `frontend/` with `npm install` and `npm run dev`

## API

The frontend reads data from the backend endpoint at `http://localhost:3001/api/categories`.
